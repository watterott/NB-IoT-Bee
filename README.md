# NB-IoT-Bee
XBee footprint compatible NB-IoT Module with u-blox SARA-R4 Cellular Module.

* [u-blox SARA-R410M-02B](https://www.u-blox.com/en/product/sara-r4n4-series) LTE Cat M1/NB1 Module
* Integrated IP stack
* UART interface
* XBee compatible Pinout (3.3V)
